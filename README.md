## About

**Aninamer** is a small, user-friendly Windows app written in C# that lets you **rename batches of anime episode files** to a clean, ordered naming scheme for use with media servers like **Jellyfin**.  
It reads file lists, matches episodes to index items, and performs safe renames.

> A simple utility to organize messy anime files into proper episode names for Plex/Jellyfin libraries.

---

## Features

- Bulk file renaming with index mapping
- Sanitizes illegal Windows filename characters
- Matches season/episode identifiers from metadata
- Saves processing logs and previews
- Windows .NET UI with easy interaction
